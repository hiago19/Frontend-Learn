Parágrafos e quebras de linhas

Você pode escrever um paragrafo de qualquer jeito. Contudo, é importante que você saiba que o HTML não leva em consideração a quantidade de espaços que você coloca entre as palavras.

Para quebrar uma linha, você pode usar a tag "<br>"
ou então, você pode usar a tag "<p>"que é a tag de parágrafo.
